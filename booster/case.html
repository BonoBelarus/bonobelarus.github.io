Задачи

  • Обеспечить технологическую линию по производству батончиков-мюсли подачей пара .

  • Установить парогенератор без здания котельной в производственном цеху в непосредственной близости с линией, для быстрой подачи пара.

  • Не иметь ограничения по пуску/останову в день.

  • Установить парогенератор без здания котельной в производственном цеху в непосредственной близости с линией, для быстрой подачи пара.

  • Не иметь ограничения по пуску/останову в день.

Решение

Установили прямоточный водотрубный парогенератор BOOSTER BOP-600G производительностью 600 кг пара в час с 2-ух ступенчатой газовой горелкой и экономайзером для увеличения КПД.

В виду своей конструкции, данный тип парогенератора можно устанавливать в производственном цеху возле потребителя пара.

Это позволяет существенно сэкономить средства на возведении здания котельной, а так же на транспортировке пара по магистралям.
